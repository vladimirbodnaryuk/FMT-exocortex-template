# Генерация обложек для постов (S48)

Автоматическая генерация cover-изображений для постов из индекса знаний через OpenAI GPT Image API.

## Что делает

Читает `.md`-файл поста → извлекает заголовок, текст, теги, аудиторию → строит SOTA-промпт → генерирует уникальную обложку, отражающую содержание статьи → сохраняет `cover.png` рядом с постом.

## Требования

- **Python 3.10+** с `httpx` и `pyyaml` (`pip install httpx pyyaml`)
- **OpenAI API key** с доступом к модели `gpt-image-1`

## Установка

### 1. API key

Получите API key на [platform.openai.com](https://platform.openai.com/api-keys).

> Нужен именно API key (platform), не подписка ChatGPT Plus. Это разные продукты.

Создайте файл `.secrets/openai-api-key` в корне workspace:

```bash
echo "sk-proj-ВАШ_КЛЮЧ" > .secrets/openai-api-key
```

Убедитесь, что `.secrets/` в `.gitignore` (в шаблоне уже есть).

Или задайте через переменную окружения:

```bash
export OPENAI_API_KEY="sk-proj-ВАШ_КЛЮЧ"
```

### 2. Проверка

```bash
python setup/optional/generate-post-image.py path/to/post.md --dry-run
```

Если видите промпт — всё настроено.

## Использование

```bash
# Автоматическая генерация из содержимого поста:
python setup/optional/generate-post-image.py path/to/post.md

# С кастомной темой (если автоматический промпт не подходит):
python setup/optional/generate-post-image.py path/to/post.md --prompt "мост между двумя мирами знаний"

# Квадратная картинка (для соцсетей):
python setup/optional/generate-post-image.py path/to/post.md --size 1024x1024

# Высокое качество (дороже, ~$0.16 вместо ~$0.07):
python setup/optional/generate-post-image.py path/to/post.md --quality high

# Dry-run (показать промпт без генерации):
python setup/optional/generate-post-image.py path/to/post.md --dry-run
```

## Как работает промпт

Скрипт строит SOTA-промпт из содержимого поста:

1. **Текст статьи** (до 2000 символов, 8 абзацев) — модель читает содержание и находит визуальную метафору
2. **Настроение** — определяется по аудитории:
   - `wide` → тёплое, приглашающее (как рассвет над новым путём)
   - `community` → сфокусированное, практичное (как мастерская с инструментами)
   - `advanced` → точное, техническое (как чертёж под направленным светом)
3. **Стиль** — editorial illustration (как обложка журнала)
4. **Композиция** — широкий кинематографический кадр, правило третей
5. **Правила** — никакого текста на картинке, метафора вместо буквального изображения

### Пример промпта

Для поста «Извозчики XXI века: почему ИТ-шники не замечают, что меняется не технология, а культура»:

> Модель получает полный текст про извозчиков 1900 года, автомобили, и параллель с разработчиками → генерирует: извозчик с лошадью смотрит на уезжающий автомобиль с AI-символом на закате.

## Параметры

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| `--size` | `1536x1024` | Размер: `1536x1024` (горизонтальная), `1024x1024` (квадрат), `1024x1536` (вертикальная) |
| `--quality` | `medium` | Качество: `low` (~$0.02), `medium` (~$0.07), `high` (~$0.16) |
| `--prompt` | из поста | Кастомный промпт (вместо автоматического) |
| `--output` | рядом с постом | Путь сохранения |
| `--dry-run` | — | Показать промпт без генерации |

## Стоимость

~$0.07 за картинку (medium quality, 1536x1024). Для 10 постов в месяц — ~$0.70.

## Интеграция с публикаторами

Если ваш контент-конвейер поддерживает автопубликацию (бот, скрипты) — публикатор может автоматически подхватывать `cover.png` рядом с постом и вставлять в начало поста при публикации.
