# WP-REGISTRY — Реестр всех рабочих продуктов

> **Сортировка:** от последнего к первому.
> **Обновляет:** R6 Кодировщик → Close, Шаг 2 (одновременно с MEMORY.md).
> **Источники:** MEMORY.md · WeekPlan W{N} · inbox/WP-{N}-*.md · archive/wp-contexts/
> **Spec:** DP.WP.015

| Статус | Расшифровка |
|--------|-------------|
| ✅ | done |
| 🔄 | in_progress |
| ⏳ | pending |
| 📦 | archived / → MAPSTRATEGIC |
| ↗️ | merged в другой РП |

---

**Колонка «Активация»** — условие, при котором pending-РП попадёт в WeekPlan:
- **date:** конкретная неделя/дата (`W15`, `после 1 апр`)
- **dep:** зависимость от другого РП (`dep: WP-3`)
- **on-demand:** только вручную на стратегировании (+ дата создания для Dormant Review)
- `on-demand` старше 3 недель → **Dormant Review** на стратегировании: архивировать (📦) или назначить условие.
