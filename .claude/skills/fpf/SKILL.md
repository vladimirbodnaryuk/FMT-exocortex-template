# Загрузка принципов

Запрос: $ARGUMENTS

## Иерархия источников (Fallback Chain)

```
1. Pack'и → PACK-* (предметные знания, 2-й уровень)
2. SPF → SPF/ (фреймворк корректности, 2-й уровень)
3. FPF → FPF/FPF-Spec.md (первые принципы, 1-й уровень — только если нет в SPF)
```

## Алгоритм

1. **Определи уровень** для задачи:
   - Предметное знание (архитектура, домен) → Pack
   - Форма, процесс, корректность → SPF
   - Базовые различения → FPF (через SPF)

2. **Загрузи принципы** из нужного источника

3. **Если нужен FPF напрямую** (редко):
   - Part A (634-16099) — роли, системы, архитектура
   - Part B (16100-20246) — рассуждения, доверие
   - Part G (39501-42007) — SoTA-методы

4. **Покажи** релевантные принципы пользователю с пояснением, как они применимы к текущей задаче
